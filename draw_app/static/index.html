    Phrases the parser understands

    • fillet all edges 2mm · round the corners 3mm
    • chamfer 1mm on the bottom · bevel 0.5mm
    • 5mm hole at center · 6mm hole at top right
    • make it 30mm tall · height 15mm
    • mirror · mirror across XZ
    • pattern 4 along x every 20mm
    • circular pattern 6 around 25mm
    • hollow 2mm wall · shell 1.5mm open top

    Anything else gets handed to Claude (if your API key has credits) — the LLM fallback converts free-form English to structured ops.

    Sketch click-actions

    • Click on/near a stroke → inline input opens anchored to it. Type ⌀10mm, width 30mm, etc. The drawing rescales.
    • Click empty canvas → inline input opens as a global command.
